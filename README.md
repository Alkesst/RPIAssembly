# RPISimonAssembly

### Requierements

  - [WiringPi][1]

### Usage
For compile and link with all the libraries you can use compileAndLink.sh. The script will create the dicerollSimulator.o and will link it to libberry.o and wiringPi. 
```bash
./compileAndLink.sh diceRollSimulator
```
